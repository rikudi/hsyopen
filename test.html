Guide to Add a Remote Repository

  1. Create a new repository on GitHub (or any other Git hosting service).
  2. Copy the repository URL (e.g., https://github.com/username/repository.git).
  3. Open your terminal and navigate to your local repository.
  4. Add the remote repository using the following command: 
    git remote add origin https://github.com/username/repository.git
  5. Verify the new remote URL: 
    git remote -v
  6. Push your changes to the remote repository: 
    git push -u origin master
